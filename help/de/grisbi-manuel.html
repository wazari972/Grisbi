Seit der Version 0.4.1 ist das Handbuch kein fixer Bestandteil von einer Grisbi-Basisinstallation.
Die Gründe dafür sind, die laufende Weiterentwicklung und die Mehrsprachigkeit.

Verfügbare Versionen

Das Handbuch ist in zwei Versionen verfügbar:

  • eine reine Textversion
  • eine Bild und Textversion

Die zweite Version ist etwas benutzerfreundlicher, jedoch dauert der Download länger und das Handbuch belegt mehr Platz auf der Festplatte.

Das Handbuch installieren

Das Handbuch kann von der Grisbi-Webseite unter Dokumentation heruntergeladen werden. Aktuell gibt es nur eine französische Version vom Handbuch. Wenn Sie das Handbuch in Ihre Sprache übersetzen wollen, kündigen Sie dies bitte in der Mailing-Liste für Entwickler an.

Handbuch mit Softwarepaket installieren

Das Handbuch ist wie das Programm selbst als RPM-Paket verfügbar.
Wenn Ihre Distribution keine Pakteinstallation unterstützt, lesen Sie bitte den Abschnitt 'Manuelle Installation'.
Die Dateien des Handbuches werden im Verzeichnis /usr/share/doc/grisbi/help/de installiert.

Handbuch manuell installieren

Wenn eine Installation mit einem Softwarepaket nicht möglich ist, oder wenn Sie es bevorzugen das Handbuch manuell zu installieren, müssen folgende Schritte ausgeführt werden:

  1. Laden Sie das Handbuch [manuel-grisbi-xxx-n.n.n.tar.gz] herunter
  2. Entpacken Sie das Archiv in das Verzeichnis /usr/share/doc/grisbi/help/de
