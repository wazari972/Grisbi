Chapitre 1  Création du fichier de comptes

1.1  Importer des fichiers QIF

Nous supposerons que vous avez exporté les comptes de votre logiciel prédédent au format QIF (attention, il vous faut un fichier QIF par compte). Pour importer ces comptes dans Grisbi :
  • cliquez sur Fichier/Importer/Fichier QIF ;

  • sélectionnez le répertoire où se trouvent ces fichiers QIF ;

  • sélectionnez-les tous en cliquant sur leur nom. Grisbi doit les importer simultanément pour reconstituer les liens entre les comptes (virements notamment). Validez. Grisbi vous demandant une confirmation, vous pouvez rajouter un compte si vous en avez oublié ;

  • choisissez la monnaie de chaque compte. Par défaut, vous avez le choix entre Francs et Euros, mais vous pouvez créer une autre devise.
Grisbi travaille quelques instants, et c'est fini.

1.2  Nouveau fichier

Vous n'avez pas de fichier à importer. Cliquez sur Fichier/Nouveau pour créer un nouveau fichier de comptes puis sur Compte/Nouveau pour créer les comptes nécessaires.

Cliquez ensuite sur l'onglet Opérations, puis sur l'icône de gauche Affiche/cache le formulaire.

Vous êtes prêt à saisir vos opérations.

Chapitre 2  Les onglets de Grisbi

2.1  Accueil

Présente la liste de vos comptes, leur solde, les échéances et les alertes.

2.2  Opérations

Présente dans sa partie droite les opérations du compte que vous avez sélectionné dans la partie gauche de l'onglet et permet de saisir les opérations. Affichez le formulaire de saisie en cliquant sur l'icône de gauche ou en double-cliquant dans une ligne vide de la liste des opérations.

Affiche les opérations de façon plus ou moins détaillée, et en masquant ou non les opérations déjà rapprochées (deux groupes d'icônes permettent d'inverser les affichages).

Pour créer un tiers n'existant pas dans la base, tapez simplement son nom dans la zone de saisie, il sera ajouté à la base.

Pour créer une catégorie faites de même, mais attention à la syntaxe : séparez le nom de la catégorie et celui de la sous-catégorie par :
  • un espace ;

  • deux points (:) ;

  • un autre espace,
sinon Grisbi ne comprendra pas.

Pour créer une imputation budgétaire c'est la même chose que pour les catégories.

Pour les exercices, allez faire un tour dans le menu Configuration/Exercices.

2.3  Échancier

Permet de programmer des opérations périodiques.

2.4  Comptes

Présente la liste de tous les comptes et leurs propriétés, et permet leur modification.

2.5  Tiers

Affiche la liste de tous vos tiers, des informations complémentaires les concernant, et la liste de leurs opérations classées par compte. Permet d'accéder immédiatement à une opération avec un double-clic. Permet également de modifier le nom du tiers.

2.6  Catégories

Identique à l'onglet Tiers, il permet de gérer vos catégories de recette et de dépense.

2.7  Imputations budgétaires

Identique aux deux précédents, il permet de gérer vos imputations budgétaires.

2.8  États

Permet de créer tous les états que vous voulez. Un certain nombre d'états sont déjà pré-formatés, et vous disposez de possibilités de personnalisation très performantes. Il n'est pour l'instant pas possible d'imprimer ces états (prévu pour la version 0.4.3).

Chapitre 3  Les menus de Grisbi

Les menus Fichier et Compte sont assez intuitifs. Le menu Configuration permet de configurer le comportement général de Grisbi et est assez parlant également.
